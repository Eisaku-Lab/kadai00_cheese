### 学んだこと・気づき
- webデザインの流行性
 - 他社サイトを参考にして視覚的な区分けや影を反映。調べたところ、影の表現は2015年ごろから出てきた“マテリアルデザイン“という様式に通じるらしい。
 - webデザインには時代性や流行性があり、これからも変遷していくこと・サイト運営者も変化に対応していく必要性があることを理解。
webデザインと信頼性への影響
制作初期、テキストの羅列だけの状態が古いサイトに見え、現代のサービスサイトとしては不適切で信用ならない印象を受ける。これはすなわち、webデザインの品質がサービスや運営会社への信頼性に影響していることだと実感。
例えば、デザインの古さ・統一感のなさ・ユーザビリティの悪さ・リンク切れなど、良いデザインが加点されるというより、問題があると信頼を損なう減点方式の傾向が強そうである。
